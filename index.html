2. Create Your App

A. Open Terminal

On Mac: Open Spotlight (Cmd + Space), type "Terminal"

6. Deploy to the Internet

A. Install gh-pages

npm install gh-pages --save-dev

B. Update package.json

Add these lines (replace YOUR-USERNAME):

"homepage": "https://YOUR-USERNAME.github.io/my-ai-app",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d build"
}

C. Deploy!

npm run deploy

Wait a few minutes, then visit:

https://YOUR-USERNAME.github.io/my-ai-app
